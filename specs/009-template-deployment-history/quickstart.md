# Quickstart: Deployment History

## Viewing the History
1. Log in to the Hermes CMS Admin Panel.
2. In the sidebar, expand the **Templates** section.
3. Click on **Deployment History**.
4. You will see a list of all template deployments for your active tenant.

## Features
- **Search**: Type in the search box to filter by template name.
- **Filter**: Use the chips to filter by 'All', 'Success', or 'Failed'.
- **Pagination**: Navigate through multiple pages using the pagination bar at the bottom.
- **Detail**: Click on a row to see the full deployment log.
